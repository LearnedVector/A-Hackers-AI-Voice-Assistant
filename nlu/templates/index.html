Intent understanding
API

This is a high level API for building voice assistants, chatbots, and more. The NLP models in this server will handle your queries and understand the intent of your question, and return an appropriate response to your client handle script. All you need to do is to register an API key and use the 'client.py' script to send and recieve data from our servers.
